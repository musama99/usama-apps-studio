# CricScore Google Play Publishing Checklist

## Privacy and Data Safety

- [ ] Host and test: `https://musama99.github.io/usama-apps/privacy/cricscore.html`
- [ ] Confirm the page opens publicly without login or geographic restriction.
- [ ] Confirm the app name and developer name match the Play Store listing.
- [ ] Confirm the final AAB contains no undeclared analytics, ads, crash reporting, tracking, or cloud SDKs.
- [ ] Review the final Android manifest permissions and match them to the policy.
- [ ] Complete the Play Console Data Safety form even when no user data is collected off-device.
- [ ] Declare no account creation only if the final app truly has no registration or sign-in flow.
- [ ] Update the policy and Data Safety form before adding any SDK or network feature.

## Release checks

- [ ] Test install, update, uninstall, clear-storage, PDF export, image export, and sharing behavior.
- [ ] Test without internet access.
- [ ] Test on the minimum and target Android versions.
- [ ] Verify all user-facing support and privacy links.
- [ ] Verify version name and version code in the final AAB.
